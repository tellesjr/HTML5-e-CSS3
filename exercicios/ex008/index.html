Negrigo / Destaque

Nesta frase temos um termo em negrito usando a tag <b> NÃO Semântica (forma sem sentido).

Nesta frase temos um termo em destaque usando a tag <strong> Semântica (que tem um sentido).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> NÃO Semântica (apenas forma sem sentido)

Texto Marcado

podemos criar também texto marcado usando a tag <mark>

E neste paragráfo temos outro texto marcado.

Perceba que no paragrafo acima o texto marcado não herdou o estilo do outro, pois alteramos apenas o primeiro

Se quiser tornar a alterção de estilo padrão no documento, você deve desenvolver as CSS na parte da cabeça (head) usando a tag <style>, declarando a tag mark (sem os couxetes angulares) abra e fecha chaves {} e dentro dos seletores use background-color: magenta; (exemplo).

Observe que agora toda vez que a tag MARK for invocada ela se apresentará na cor de fundo magenta.

Texto grande e pequeno

temos aqui um (texto grande) usando a tag <big>. Observação esta tag esta obsoleta

Neste parágrafo temos um (texto pequeno) usando a tag <small>. Esta tag ainda esta em uso no HTML5

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser ligo, mas não considerado, use a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, use a tag <ins>

Existe também o Sublinhado com a tag <u> (Não Semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3. use a tag <sup>.

Texto Subscrito

Para inserir coisas do tipó H2O. use a tag <sub>.
