Usando links externos

para colocar um link no seu site use a tag <a>

Exemplos:

<p>Você pode acessar meu <a href="https://gustavoguanabara.github.io">repositorio no gitHub</a></p>

Você pode acessar meu repositorio no gitHub

Você também pode acessar o meu canal gratuito no youtube.

Outro

Use a tag <a> com os parametros (target="_blank" E rel="external")
