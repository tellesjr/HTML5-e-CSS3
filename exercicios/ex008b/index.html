Codigo-Fonte / Pré-Formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Perceba como é dificil ler o codigo acima com a fonte padrão. veja abaixo o mesmo codigo envelopado com a tag <code>

document.getElementById('teste')

Usamos essa tag para deixar a fonte mono-espaçada.

Veja abaixo um codigo em linguagem Python apenas com a tag <code>

num = imput(int('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR) else: print(f'O número {num} é IMPAR) print('fim do programa')

Perceba que o código não ficou com uma estrutura de alinhamento de código. Para resolver esse problema podemos usar a tag <pre> envelopando a tag <code>. Veja abaixo.

        
    num = imput(int('Digite um número'))
    if num % 2 == 0:
    print(f'O número {num} é PAR)
    else:
        print(f'O número {num} é IMPAR)
    print('fim do programa')

Usando essa tag você consegue conservar a estrutura e alinhamento dos códigos ou qualquer outro texto que você queira que fique examente como escreveu, com os estapaços e alinhamentos. Você pode tambem usar as teclas CTRL+TAB ou TAB para acrescentar ou diminuir a tabulação.
